# ✨ Clipboard Landing Page

![](./design/preview.jpg)

## 📖 Description

Clipboard landing page developed with HTML, CSS, JavaScript, and the Less preprocessor.

## 📦 Install and usage

1. Clone this repo:

```bash
git clone https://github.com/GracilianoOG/fem-clipboard-landing-page.git
```

2. Enter the directory:

```bash
cd fem-clipboard-landing-page/
```

3. Init Visual Studio Code (or any other IDE):

```bash
code .
```

4. Install the dependencies:

```bash
npm install
```

5. Run the project:

```bash
npm run dev
```

## 📜 Available scripts

| Script            | Description                            |
| ----------------- | -------------------------------------- |
| `npm run dev`     | Runs the development server.           |
| `npm run build`   | Builds the project.                    |
| `npm run preview` | Previews the production build locally. |
